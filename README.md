## Getting Started

Follow these steps to set up and run the project locally.

### 1. Clone the Repository
```bash
git clone https://github.com/KeylaNyacyesa/alu_pulse.git
cd alu_pulse
```

### 2. Install Dependencies
Fetch all the required packages listed in the `pubspec.yaml` file:
```bash
flutter pub get
```

### 3. Verify Environment
Run the Flutter diagnostic tool to ensure your setup is correct and a device is detected:
```bash
flutter doctor
```

### 4. Run the Application
Launch the app on your connected device or emulator:
```bash
flutter run
```
